### 💻 Projetos em Destaque

* **[ClinPlaY](https://front-clin-play.vercel.app)**: Aplicativo web e mobile gamificado projetado para auxiliar pacientes e clínicas de fisioterapia pélvica. Atuei como Líder Técnico na modelagem arquitetural, estruturação do software e coordenação do desenvolvimento fullstack.
* **[El Banquero](https://github.com/catmaitachi/El_Banqueiro)**: Projeto lógico focado em sistemas operacionais. Implementação em Python do Algoritmo do Banqueiro (*Banker's Algorithm*) para simular, analisar e gerenciar com precisão a prevenção de *deadlocks* na alocação de recursos.

Desenvolvido para integrar e automatizar a regra de mecânica opcional de Morte Alternativa do RPG de mesa Tormenta20, manipulando condicionais da API do Foundry e atualizando estados dinâmicos de componentes em tempo real.
